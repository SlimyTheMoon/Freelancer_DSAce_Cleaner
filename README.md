# Discovery Freelancer RP Log Editor & Filter

A lightweight, browser-based tool designed to clean up and filter roleplay (RP) chat logs from the Discovery Freelancer 24/7 RP server. It helps players quickly extract specific RP interactions from massive raw log files (like `DSAce.log`) for archiving, storytelling, or posting on the forums.

## ✨ Features

* **100% Local & Secure:** Runs entirely in your web browser. No servers, no installations, and your logs are never uploaded to the internet.
* **Upload or Paste:** Paste your raw logs directly into the tool, or upload a `.txt` file straight from your Freelancer directory.
* **Smart System Message Removal:** Automatically strips out server rules, traffic alerts, death messages, base bot automated messages, and generic console spam so you can focus on the roleplay.
* **Player Whitelisting:** Enter a comma-separated list of character names to extract *only* the conversations involving those specific players.
* **Date Filtering:** Set a start and end date to narrow down massive log files to a single day or event.
* **Timestamp Cropping:** Optionally strip the `[DD.MM.YYYY HH:MM:SS]` timestamp prefixes for a clean, dialogue-only reading experience.
* **Dark Mode UI:** A sleek, dark-themed interface that is easy on the eyes.

## 🚀 How to Use

1. Download or save the `index.html` file to your computer.
2. Double-click `index.html` to open it in your preferred web browser (Chrome, Firefox, Edge, Safari, etc.).
3. **Input Logs:** Paste your copied log text into the top box, or click **Choose File** to upload a `.txt` log file (e.g., `DSAce - Copy (2).log`).
4. **Configure Filters:**
   * **Name Whitelist:** Type the names of the characters you want to keep (e.g., `Velvet.Knife, Oder_Supply`). Leave blank to keep all player messages.
   * **Dates:** Select a specific time range using the Start Date and End Date calendars.
   * **Checkboxes:** Check the boxes to remove system clutter and/or cut timestamps.
5. Click the **Process Logs** button.
6. Your cleaned, filtered roleplay logs will instantly appear in the bottom Output box, ready to be copied!

## ⚙️ How the Filters Work

* **System Messages:** The tool looks for specific keywords common to Discovery GC (e.g., `Traffic control alert`, `Rules Tip:`, `Death:`, `Auto-Buy`, `Welcome to the Discovery Freelancer 24/7 RP server!`, etc.) and completely removes those lines. It also filters out raw command inputs (like `/stuck` or `/time`).
* **Whitelisting:** The tool scans the sender portion of the line. If you put `[EX]-TheSparrow` in the whitelist, it will keep lines where that character is speaking.
* **Date Parsing:** Extracts the standard `[DD.MM.YYYY]` format generated by DSAce and compares it against your selected calendar range.

## 🛠️ Technical Details

This project is built as a **Single-Page Application (SPA)** contained entirely within one `index.html` file. It uses standard HTML5, CSS3, and Vanilla JavaScript. There are no external dependencies or frameworks required.
